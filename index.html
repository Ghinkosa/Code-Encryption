Protect your PHP & Laravel source code

Galata Code Protector encrypts PHP files (AES-256) and loads them securely at runtime using a lightweight loader. Ship closed-source builds to support teams without exposing your code.

AES-256 Encryption

Each PHP file is encrypted with AES-256-CBC; payloads stored as .galo.

Runtime Loader

Stubs decrypt code in memory and execute without creating decrypted files on disk.

Laravel Friendly

Composer package, Artisan command, and safe default exclusions for kernel, providers and bootstrap.

Cross-platform

Path normalization and skip rules make the tool work on Windows, macOS and Linux.

Safe Stubs

Stubs contain no business logic — they only trigger loader execution.

CLI Workflow

Encrypt your project via php artisan code:encrypt and configure via config/codeprotect.php.

Install

Install the package in your Laravel project and publish the config file. Use Composer for Packagist install or a local path repository for development.

composer require galata-eth/laravel-code-protector
php artisan vendor:publish --tag=config
# set CODE_PROTECT_KEY in .env
php artisan code:encrypt

⚠️ Always backup your source before running encryption.

Quick Config

return [
  'key' => env('CODE_PROTECT_KEY', env('APP_KEY')),
  'paths' => ['app/'],
  'enc_suffix' => '.galo',
];

Usage & Examples

Encrypt a single folder

php artisan code:encrypt --path=app/Services

Generate key (example)

<?php echo base64_encode(random_bytes(32)); ?>

Stub example (generated)

<?php
// Encrypted stub
\GalataEth\CodeProtect\Loader::loadFromStub(__FILE__);

Documentation

Config Options

  • key — encryption key (base64:... or APP_KEY)
  • paths — array of folders to encrypt
  • enc_suffix — suffix for encrypted payloads

Recommended Excludes

Always exclude files that run during bootstrap:

app/Console/Kernel.php
app/Http/Kernel.php
app/Providers/AppServiceProvider.php
bootstrap/app.php
vendor/
config/
resources/views

FAQ

Will encrypted files break my app?

If you keep the recommended excludes (kernel, providers, bootstrap) and test locally, your app will run normally.

Can I decrypt files?

Only with the same encryption key. Keep backups of originals.
